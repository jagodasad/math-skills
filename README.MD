math-skills is a program that is given at audit to test if numbers are correct.

how to run:
1. Download the file from the audit questionnaire I will send you or here is the link to download the file https://assets.01-edu.org/stats-projects/math-skills
2. Copy it onto the folder math-skills you just cloned
3. Rename it to math-skills.sh
4. Give it permission by typing chmod 777 math-skills.sh
5. Run on your terminal ./math-skills.sh
6. Run on your terminal go run . data.txt
7. You should see the same results of average, median variance and standard deviation when comparing.
